# wind

介绍: 柜柜小渲风官网 

## 1. 技术栈介绍

vue2 + ts + nuxt.js + elementui

## 2. 运行指令

- 安装依赖：npm i
- 项目运行: npm run serve
- 打包(正式服): npm run build
- 打包(测试环境): npm run build:test
- 打包(开发环境): npm run build:dev

## 3. 代码格式化规范

1. 关闭 git 的回车符自动转换: git config --global core.autocrlf input
2. 安装 prettier 插件
3. 安装 eslint 插件
4. 安装 editorConfig 插件
